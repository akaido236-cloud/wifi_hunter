#### 1. **ARP Spoofing الاحترافي**
- تحويل حركة المسار الذكي

#### 2. **DNS Sniffer عالي الأداء**
- التقاط جميع استعلامات DNS

- تسجيل الاستعلامات الحية

#### 3. **DNS Cache Poisoning** 🔴
- حقن عناوين IP وهمية
- تحويل حركة المستخدم
- تأثير فوري على التصفح

#### 4. **كشف الأجهزة المحمولة** 📱
- تحديد iPhone/iPad تلقائياً
- كشف أجهزة Android
- قائمة الأجهزة المكتشفة

#### 5. **استخراج بيانات الاعتماد** 🔐
- البحث عن كلمات المرور
- استخراج بيانات المستخدم
- تسجيل في ملف منفصل

#### 6. **تحليل حركة HTTP/HTTPS** 🌐
- عد طلبات HTTP و HTTPS
- تتبع الجلسات
- حساب البيانات المنقولة

#### 7. **حجب المجالات** 🚫
- قائمة مجالات محجوبة قابلة للتخصيص
- إعادة توجيه تلقائي
- تسجيل محاولات الوصول

#### 8. **نظام السجلات المتقدم** 📁
- `dns_log_*.txt` - استعلامات DNS
- `sessions_*.json` - الجلسات النشطة
- `credentials_*.txt` - بيانات الاعتماد
- `stats_*.json` - الإحصائيات الكاملة

#### 9. **الإحصائيات الشاملة** 📊
```
- إجمالي الرزم المنقولة
- البيانات بالكيلوبايت
- عناوين IP الفريدة
- أفضل 5 استعلامات DNS
- الأجهزة المحمولة المكتشفة
- بيانات الاعتماد المنقولة
```

### استخدام سريع:
